# Diva Portfolio

Personal portfolio site for **Diwakar Venkat Peddada** — Software Engineer, AI/ML developer, and research-focused builder.

**Live site:** [myselfdiva.netlify.app](https://myselfdiva.netlify.app)

## Stack

- HTML5, CSS3, vanilla JavaScript
- Three.js (WebGL floating-lines background)
- Canvas-based chroma-key video rendering

## Local development

Serve the folder with any static file server:

```bash
npx serve .
```

Or open `index.html` directly (some features work best over HTTP).

## Project structure

```
├── index.html      # Main page
├── style.css       # Styles and animations
├── script.js       # Interactions, WebGL, video effects
├── favicon.svg     # Site icon
└── assets/
    ├── chroma-keyed-video.webm
    └── download.wav
```
